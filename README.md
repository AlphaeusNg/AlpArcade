# 🕹️ AlpArcade

Browser mini-games by **Alphaeus Ng** — local scoreboard, sound, powerups, and difficulty ramps.

**Play:** https://alphaeusng.github.io/AlpArcade/

**Portfolio:** https://alphaeusng.github.io/

### Games
Tic-Tac-Toe · Space Shooter · Snake · Reaction Lab · Memory Match

### Scoreboard
Player tag, XP, personal bests, hall of fame — stored in `localStorage` with export/import codes.

### Enable GitHub Pages
Repo **Settings → Pages → Deploy from branch → `main` / root**  
(Remove any broken custom domain so the free `github.io` URL works.)
